A Discord BOT called YurroBot that has been created to serve the SpaceAddicts Discord community. Anyone in the community can call on YurroBot for any character profile by simply typing "!lookup nameofcharacter".
All characters are based on the NFT comic creation Space Addicts ( http://spaceaddicts.io)

All this code belongs to the Space Addicts NFT group and can be used freely.

=-=-=-=-=-=-Commands-=-=-=-=-=

!yurro help - This will show all the commands for Yurrobot

!lookup - Typing this followed by a character name will give you a description of the character lore.

!lookupnft - Typing this followed by a number will return the image of the NFT.

!lookuptraits - Typing this followed by a number will return all the traits for that particular NFT.

!random - This will return a random character profile from the SpaceAddicts lore.
